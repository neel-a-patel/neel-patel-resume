## 🚀 Quick Start (5 Steps)

### Step 1: Install Node.js
- Download from [nodejs.org](https://nodejs.org)
- Install (includes npm)
- Verify: `node --version` and `npm --version`

### Step 2: Install Dependencies
```bash
cd neel-patel-resume
npm install
```
⏱️ Takes 2-5 minutes

### Step 3: Run Locally
```bash
npm run dev
```
🌐 Opens http://localhost:3000 automatically

### Step 4: Customize Content
- Edit files in `src/components/`
- Changes reload automatically
- See CUSTOMIZATION.md for details

### Step 5: Deploy
```bash
npm run deploy
```
🎉 Website live on GitHub Pages!

## 🆘 Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| npm command not found | Install Node.js from nodejs.org |
| Port 3000 in use | Use `npm run dev -- --port 3001` |
| Build fails | Run `npm cache clean --force && npm install` |
| Styles not showing | Clear browser cache (Ctrl+Shift+Delete) |
| Hot reload not working | Stop dev server, run `npm run dev` again |
| Git errors | Initialize git: `git init` |
| GitHub deploy fails | Verify repository and branch names |
